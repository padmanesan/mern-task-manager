Task Manager

A web application to efficiently manage tasks, track progress, and organize daily work. This project helps users create, update, delete, and view tasks in a clean and interactive interface.

Features

User Authentication: Sign up, login, and secure access to tasks.

Task Management: Create, edit, and delete tasks easily.

Task Status: Mark tasks as complete or pending.

Responsive Design: Works well on mobile, tablet, and desktop.

Backend & Database: Built with Node.js, Express.js, and MongoDB.

Tech Stack

Frontend: HTML, CSS, JavaScript, React (if used)

Backend: Node.js, Express.js

Database: MongoDB

Version Control: Git & GitHub
Installation

Clone the repository:

git clone <repo-link>


Navigate to the backend directory and install dependencies:

cd backend
npm install


Set up MongoDB connection in .env file:

MONGO_URI=your_mongodb_uri


Start the backend server:

npm start


For frontend (if React is used):

cd frontend
npm install
npm start
